# 闭口

## 区分闭口和粉刺

闭口属于粉刺的一种，叫白头粉刺。所以说，闭口=白头粉刺。

图一是白色粉刺，即闭口；图二是黑色粉刺。

它们 2 个的区别在于，黑头接触到外界空气被氧化了，所以就黑色了。而闭口之所以白色，是因为还在毛囊里面，还没接触到空气。
